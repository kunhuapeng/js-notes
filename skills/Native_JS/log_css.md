# console.log() 样式

可以使用 CSS 语句在 DevTools 中设置 `console.log` 输出的样式：

```js
console.log('%c Success', 'color:green; font-size:1.5em');

console.log('%c Warning', 'color:orange; font-size:1.5em');

console.log('%c Error', 'color:red; font-size:1.5em');
```
